# AI Ticket Support\n\nUsed for generating polite FunPay support ticket text when buyers forget to confirm orders.\n\nEnv vars:\n- GROQ_API_KEY: required for Groq API.\n- GROQ_MODEL: optional, default 'llama-3.1-8b-instant'.\n\nBehavior:\n- Bot builds a brief prompt with order id, buyer, lot.\n- If GROQ_API_KEY is missing or request fails, falls back to a static polite message.\n
